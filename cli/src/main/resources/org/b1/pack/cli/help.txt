Usage: b1 <command> <options> <archive> <files>

Commands:
 a: Add files to archive
 l: List contents of archive
 x: Extract files from archive

Options:
 -h, -?, --help: Show usage instructions
 -v <size>[B, kB, MB, GB]: Create volumes
 -m [smart, classic, maximum]: Set compression method
 --type <type>: Set type of archive
 -o <path>: Set output directory
 -p: Encrypt with a password (not implemented yet)
